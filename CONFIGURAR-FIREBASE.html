Configuración de Firebase

Esta versión ya contiene la conexión, reglas y estructura. Para activarla faltan únicamente los datos de tu propio proyecto Firebase.

1. Crear el proyecto

En Firebase Console crea un proyecto llamado, por ejemplo, rifa-con-causa.

2. Activar servicios

Activa Authentication con correo y contraseña, Cloud Firestore y Storage.

3. Crear administradora

En Authentication crea una cuenta de correo y contraseña para entrar al panel.

4. Copiar configuración

Registra una aplicación web y copia sus datos dentro de firebase/firebase-config.js.

5. Activar modo Firebase

En config.js cambia:

mode: "local"

por:

mode: "firebase"

6. Publicar reglas

firebase login
firebase use --add
firebase deploy --only firestore:rules,firestore:indexes,storage

7. Inicializar números

Entra al panel con tu correo de Firebase y pulsa Inicializar 500 números.

Importante: hasta completar estos pasos, la aplicación continuará funcionando en modo local.
