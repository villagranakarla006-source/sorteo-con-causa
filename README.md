# Rifa con Causa — Versión 6.2

Esta entrega conserva el panel local funcional y agrega la infraestructura de producción con Firebase.

## Incluye

- Firebase Authentication para administradores.
- Firestore con 500 números y participantes.
- Transacciones para evitar que dos personas aparten el mismo número.
- Firebase Storage para comprobantes.
- Sincronización pública en tiempo real.
- Panel Firebase independiente.
- Reglas de Firestore y Storage.
- Índices y configuración de Firebase Hosting.
- Modo local de respaldo.

## Acceso local

Abre `ACCESO-ADMIN.html`.

- Usuario: `admin`
- Contraseña: `Rifa2026`

## Activación en línea

Abre `CONFIGURAR-FIREBASE.html` y sigue los pasos. Debes proporcionar los datos de tu proyecto Firebase en `firebase/firebase-config.js`; esos datos no pueden generarse automáticamente sin acceso a tu cuenta.
