# Rifa con Causa — GitHub Edition V11

Sitio compatible con GitHub Pages y conectado al proyecto Firebase `rifa-con-causa-karla`.

## Enlaces

- Sitio público: `index.html`
- Consulta: `consulta.html`
- Ganador: `ganador.html`
- Panel administrativo: `admin/` o `admin.html`

## Publicación

Sube **todo el contenido de esta carpeta** a la raíz de la rama principal del repositorio. No subas únicamente el archivo ZIP ni la carpeta contenedora.

GitHub Pages debe usar la rama `main`/`Principal` y la carpeta `/(raíz)`.

## Importante

Firebase Storage no se utiliza. Los comprobantes se envían por WhatsApp y la administradora confirma manualmente el pago.
